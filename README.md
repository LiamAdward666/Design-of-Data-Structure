# 全球航班查询系统 (Global Flight Query System)

![C++](https://img.shields.io/badge/Language-C++11-blue.svg) ![DataStructure](https://img.shields.io/badge/Topic-Data_Structure-orange.svg) ![License](https://img.shields.io/badge/License-MIT-green.svg)

## 📖 项目简介
随着全球民航数据的爆发式增长，单一维度的检索已无法满足需求。本项目通过集成多种高级排序算法与图论路径规划算法，旨在展示数据结构在解决复杂工程问题中的核心作用。

## 🚀 核心逻辑与算法
本项目严格执行任务书要求，实现了多种经典算法：

### 1. 排序算法 (Sorting)
用于航班信息的动态展示与多维度检索：
* **票价排序**：冒泡排序 (Bubble Sort)
* **飞行时长排序**：快速排序 (Quick Sort)
* **准点率排序**：堆排序 (Heap Sort)
* **航班号排序**：基数排序 (Radix Sort)

### 2. 图论算法 (Graph Theory)
底层图结构满足顶点数 $V \ge 15$：
* **全路径查询**：基于深度优先搜索 (DFS) 输出两地间所有简单路径。
* **最优路径规划**：基于 Dijkstra 算法计算票价最低方案。

## 📖 功能操作指南

### ✈️ 航班查看与排序 (选项 1-5)
1. **查看列表**：输入 `1` 查看初始航班信息。
2. **多维度排序**：根据提示输入 `2-5`，系统将应用对应算法重新排列数据。

### 🗺️ 航线咨询与规划 (选项 6-7)
* **所有路径查询**：输入 `6` → 输入起点与终点 (如: `Beijing Shanghai`) → 系统输出所有可行路径及总成本。
* **最低票价导航**：输入 `7` → 输入起点与终点 → 系统通过 Dijkstra 算法给出最优转机序列。

---

## ⚠️ 异常处理与退出
* **输入校验**：若输入非 `0-7` 的指令，系统提示“无效输入”。
* **可达性检查**：若两地间无航线，输出“无可行路径”或“城市不存在”。
* **安全退出**：主菜单输入 `0` 即可释放内存并结束程序。
